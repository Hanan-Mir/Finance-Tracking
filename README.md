#FinTrack
FinTrack is a personal finance and stock tracking application that will help the users to manage there expenses and inventory of the material in which they are dealing with in an easy way.
## Features
**EmailService:**The user can send an email using emailjs service if he wishes to get contacted by the team of finTrack
**Authentication:** For authentical we have used secure login with email and password and also we have added the functionality of Google OAuth in the application.we also have added the feature of forget password were user will get the forget password link to its email and can reset the password.
**Dashboard:** It shows visual overview of total revenue that has been generated and the average sales value and total number of transactions that have been done and also the growth percentage which compare the growth with respect to the previous month.we have also used a bar graph which will show the compare the revenue weekly and also the business summary pie chart which will show the overall summary of the business.
**Transactions:** It will show value of all the sales and expenses and also a pie with all the balances and transactions for online and cash. Here user can also search for the transactions by the name and can also set the filter and can export the whole transactions data and can also add transactions weather sale or expense and can print the recipt for the transactions and can also edit the transactions.
**Inventorty:**It will show the inventory that is present with the user and will also show all the items in which the user is dealing and will show health of the stock that is available and the KPI that will show the list of products which are low in stock
**Sales:** It will show the monthly sales and overall sales balance it will show the sales that have been done in the current week and will also compare the sales montly in the year.
**Managment:** Here user can add the products in which he is dealing with all the details like current quantity, selling price,cost price, vendor name and some other details.
## Tech Stack
* **Frontend:**
**React 18/19:** Core library for building the UI and Context Api for state session across the app.

**Tailwind CSS:** For modern, utility-first styling.

**GSAP (GreenSock):** High-performance scroll-triggered animations and UI transitions.

**React Router v7:** Managing navigation and scroll restoration.

**React Toastify:** For elegant, non-blocking user notifications (like "Login Successful"). 

* **Backend/Auth:** Supabase,
  **OtherServices:** EmailJs,Recharts,Nivocharts,
